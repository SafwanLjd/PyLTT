# PyLTT
A FOSS CLI Alternative to The Official MyLTT App

_This Relies on The [MyLTT](https://github.com/SafwanLjd/MyLTT) API Helper_

## Note
This program can stop working if the MyLTT API got a major update, right now it's compatible with [MyLTT](https://play.google.com/store/apps/details?id=com.ltt)'s 1.0.46 release, which is the latest at the time of writing this

## Current Limitations

* No support for Libya Phone
* Poor support for "pay-as-you-go" packages

### [MyLTT]((https://github.com/SafwanLjd/MyLTT)) API Helper Limitations
* No support for promotions (free "gifted" packages)
* No support for the "extra packages"

There is no technical reason for these limitaions, it's possible to add proper support for all of these, and I probably will, someday; in the meantime, feel free contribute yourself, help is always appreciated :)
